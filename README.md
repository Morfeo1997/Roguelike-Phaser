# Roguelike Phaser

## Descripción corta
- Esto es un videojuego en vista cenital, creado para poder practicar con Phaser en el desarrollo de videojuegos, un jugador puede moverse en 8 direcciones, atacar y saltar con el mouse y debera derrotar a todos los enemigos posibles para poder ganar mas puntos
## Tecnologías
- React
- Phaser
- Typescript
- Tailwindcss
## Características principales
- Combate en tiempo real con puntero de ratón
- Movimiento en 8 direcciones (WASD)
- Salto direccional
- Sistema de progresión por niveles
- Enemigos cuerpo a cuerpo y a distancia
- Sistema de vida con curación y aumento de vida máxima
- Estilo roguelike con dificultad progresiva
## Controles
- WASD para moverse
- Click izquierdo para atacar
- Click derecho para saltar
- ESC para pausar el juego
## Instalación y ejecución
- Se puede instalar facilmente utilizando el comando "npm install"
- para jugar puede visitar la pagina : https://roguelike-phaser.netlify.app/  o correr el comando "npm run dev" para jugarlo de forma local
